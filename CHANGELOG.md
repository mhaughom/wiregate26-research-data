# Changelog

## 1.0.0 — 2026-07-15

- Added a reproducible four-model incident analysis with a discrete,
  sample-cadence impulse-time search.
- Replaced the legacy 2.71 m endpoint comparison with a like-for-like 2.1901 m
  touchdown comparison based on a disclosed three-metre continuation.
- Added byte-for-byte ATS-to-incident-CSV extraction and full-ball-to-kick-window
  reconstruction tools.
- Added retained-source hashes, JSON Schemas, strict validation, a locked Python
  3.12 environment and GitHub Actions reproducibility checks.
- Added citation metadata, data-use information, contribution guidance and a
  scientific-review issue form.

The original untagged publication commit remains available in Git history. Its
data files are preserved by their manifest hashes.
